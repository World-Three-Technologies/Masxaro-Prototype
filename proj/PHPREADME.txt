In order to configure the PHP project, please follow these steps:

1. Locate your Apache web root path
   LINUX: ../htdocs/
   Windows: ../www/
   
2. Create new folder w3t
sudo mkdir w3t

3. Copy php/ and config.php to your web ROOT_PATH/
